# Automatización de la API de mascotas

Este proyecto es parte del ejercicio práctico de Banco Pichincha.

## Pre-requisitos
Para montar un ambiente de desarrollo, necesitaras:

- node ^16
- npm ^8

## Instalación

1. Abrir el proyecto en un IDE de desarrollo
2. Ejecutar el siguiente comando `npm install`

## Uso

1. Ejecutar este comando para abrir el Dashboard de Cypress

   ```bash
   npm run test
   ```
   
2. En la barra de navegación, hacer click en Specs > userAPI.cy.js
3. Las pruebas comenzarán a ejecutarse
4. Esperar a que finalice la ejecución y pueda revisar los resultados
