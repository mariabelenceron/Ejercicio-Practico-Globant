# Automatización E2E Ecommerce

Este proyecto es parte del ejercicio práctico de Banco Pichincha

## Pre-requisitos
Para montar un ambiente de desarrollo, necesitaras:

- Java Development Kit (JDK) 8 o superior
- Maven para la gestión de dependencias
- Un IDE de desarrollo Java como Eclipse
- Tener descargado en el IDE TestNG. Para Eclipse (https://marketplace.eclipse.org/content/testng-eclipse/help)

## Instalación

1. Importar el proyecto en el IDE de desarrollo de preferencia

## Uso

1. Abrir el archivo testng.xml
2. Click derecho > Run As > TestNG Suite
3. Esperar a que se ejecute la prueba

## Reporte
Una vez terminada la ejecución se puede revisar el reporte generado en la carpeta reports > index.html
